# Headless CMS & AI Blog Generator

This is a Next.js (App Router) centralized Headless CMS platform for managing blog posts across multiple clients. It features automated AI content generation using OpenAI and fetching cover images using Pexels.

## Technology Stack
- Next.js 15+ (App Router)
- PostgreSQL (via Prisma v7)
- NextAuth.js
- Tailwind CSS v4
- TipTap Rich Text Editor
- External APIs: OpenAI (gpt-4o), Pexels API

## Local Development

Ensure the following variables are present in your local `.env` file:
```env
NEXTAUTH_SECRET="..."
NEXTAUTH_URL="http://localhost:3000"
DATABASE_URL="postgres://user:password@localhost:5432/dbname"
OPENAI_API_KEY="..."
PEXELS_API_KEY="..."
```

### Installation

1. Install dependencies:
```bash
npm install
```

2. Format Prisma Client:
```bash
npx prisma generate
```

3. Deploy local database schemas:
```bash
npx prisma migrate dev
```

4. Seed the default admin user:
```bash
npm run seed
```

5. Start the development server:
```bash
npm run dev
```

Admin credentials generated by seed:
- **Email:** admin@blogcms.com
- **Password:** admin123

## Production Deployment (Vercel)

This application is ready to be hosted on Vercel. 
During the setup on Vercel:
1. Attach a **Vercel Postgres Database**.
2. Add your `OPENAI_API_KEY`, `PEXELS_API_KEY`, `NEXTAUTH_SECRET` and `NEXTAUTH_URL` securely into Vercel Settings > Environment Variables.
3. Don't forget to run `npx prisma migrate deploy` and `npm run seed` pointing to your Vercel database.

## Client Website Integration

Retrieve the generated API token per client from your Admin Panel > Clients section. Use it as a Bearer token or URL parameter to load content safely on your client projects framework (Next.js/React).

**API Route:**
```
GET /api/external/v1/posts?page=1&limit=10&slug=post-slug-optional
Headers: Authorization: Bearer <API_TOKEN>
```
